🚀 About

MenuSync AI is a modern SaaS web application that helps restaurants manage and update their menus from a single dashboard. It simplifies operations by allowing users to edit once and simulate publishing across multiple platforms.

✨ Features

  • Centralized menu management
  • Add, edit, and delete food items
  • Multi-platform menu simulation
  • User authentication system
  • Real-time backend integration

🛠 Tech Stack

  • Frontend: HTML, CSS, JavaScript / React
  • Backend: Supabase / Lovable Cloud
  • Hosting: Vercel
