## ✨ Features

| Feature | Description |
|---------|-------------|
| 📝 **Centralized Menu Management** | Manage all your restaurant menus from one intuitive dashboard |
| ➕ **CRUD Operations** | Add, edit, and delete food items with ease |
| 🌐 **Multi-Platform Simulation** | Preview how your menu looks across different platforms |
| 🔐 **User Authentication** | Secure login and registration system |
| ⚡ **Real-time Updates** | Changes sync instantly across all connected platforms |

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- npm or bun

### Installation

```bash
# Clone the repository
git clone <repo-url>

# Navigate to project
cd menu-sync

# Install dependencies
npm install

# Start development server
npm run dev
```
