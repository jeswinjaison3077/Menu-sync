# MenuSync AI

A SaaS web app for managing and updating restaurant menus across multiple platforms from a single dashboard.

## 🚀 Features
- Centralized menu management
- Add, edit, delete food items
- Multi-platform menu simulation
- User login and registration

## 🌐 Live Demo
https://menu-sync-nine.vercel.app/

## 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Supabase / Lovable
- Hosting: Vercel
